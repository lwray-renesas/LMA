◆ LMA_PhaseShift90()

spl_t LMA_PhaseShift90 ( spl_t new_voltage)

Phase shifts voltage signal.

  • 50Hz signal is 20ms.
  • 50Hz signal being 360degree of period, to get 90degree we divide by 4.
  • 20ms divided by 4 = 5ms.
  • to delay 5ms with a 3906Hz clock we can do 0.005/(1/3906) = 19.53 samples - so we do 20 samples.
Parameters
[in]new_voltage- new voltage to store in the buffer
Returns
voltage sample 90degree (20 samples) ago.
