Introduction

Welcome to the LMA homepage, here we document a lightweight AC metrology framework written in C, the aims of which are functionality, education and portability - in that order.

The quickest way to get started is to identify one of the examples in the repositories examples folder and start running it.
Each example comes equipped with a RUNME.md which contains instructions for running the example project - so head over there, choose an example & follow the respective instructions to run it.
Link to repository: https://github.com/lwray-renesas/LMA
